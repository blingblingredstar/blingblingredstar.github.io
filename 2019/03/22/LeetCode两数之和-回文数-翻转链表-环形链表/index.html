LeetCode两数之和,翻转链表

发表于 2019-03-22 更新于 2019-04-29 分类于 LeetCode

两数之和

第一种解法

直接双重循环，把所有数保存起来枚举一遍。时间复杂度O(n2)

第二种

把数组保存起来，遍历所有数，查找有没有与目标与当前遍历的数的差相等的情况。

翻转链表

创建一个空的前一节点，空的下一个节，一个当前节点

  • 保存当前节点的下一个节点
  • 当前节点的指针指向前一节点
  • 前一节点赋值为当前节点
  • 当前节点重新赋值为下一节点
