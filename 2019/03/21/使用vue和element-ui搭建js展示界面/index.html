使用vue和element-ui搭建js展示界面

发表于 2019-03-21 更新于 2023-05-17 分类于 经验心得

在知乎上看到了豪情前端作品列表觉得可以搞一个放到简历网页及GitHub上，顺便做个js项目的线上存储。之前一直没上传到GitHub，就是感觉太零碎。不过又觉得这个首页过于简单。就想到了使用vue和UI框架做一个类似后台管理界面的展示页。

技术栈决定就是Vue全家桶了。毕竟目前为止只用Vue做过东西。UI框架之前做小程序的时候发现饿了么平台出过一个element-ui，感觉不错。

参考资料
Vue.js 渐进式JavaScript框架
Element，一套为开发者、设计师和产品经理准备的基于 Vue 2.0 的桌面端组件库

总之，做就是了。
